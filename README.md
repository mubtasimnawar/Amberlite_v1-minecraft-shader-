# LaptopShader

A Minecraft Iris shader built specifically for low-end laptops and integrated graphics.

## Goals

- 60 FPS on Intel UHD graphics
- Minimal FPS drops
- Better lighting than Vanilla
- Low VRAM usage
- Fast chunk loading

## Tested Hardware

✔ Intel UHD 620
✔ i5-8350U
✔ 8GB DDR4 (2400MHz Cl18)
